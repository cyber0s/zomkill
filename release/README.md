# 🌻 ZomKill v2.0

植物大战僵尸辅助工具 - 使用 Rust + egui 构建的现代化图形界面版本

## ✨ 功能特性

- 🌞 **阳光修改** - 自由修改游戏中的阳光数值
- ⚡ **无冷却功能** - 移除所有植物卡槽的冷却时间
- 🎨 **现代化界面** - 基于 egui 的美观图形化界面
- 🔄 **实时监控** - 自动检测游戏运行状态和当前阳光值
- ⚡ **快捷操作** - 一键设置常用阳光数值
- 🈯 **完美中文支持** - 内置思源黑体，中文显示完美

## 📋 系统要求

- Windows 操作系统
- 《植物大战僵尸年度版》游戏

## 🚀 使用方法

1. 启动《植物大战僵尸年度版》
2. 运行 ZomKill 程序
3. 程序会自动检测游戏并连接
4. 使用界面上的功能按钮进行操作

### 阳光修改
- 在输入框中输入想要的阳光值
- 点击"修改阳光"按钮
- 或使用快捷按钮一键设置常用数值（9990/8000/5000）

### 无冷却功能
- 点击"启动无冷却"按钮激活功能
- 所有植物卡槽将持续保持无冷却状态
- 点击"停止无冷却"可以关闭功能

## 🛠️ 编译构建

```bash
# 克隆项目
git clone <repository_url>
cd zomkill

# 编译运行
cargo run --release

# 仅编译
cargo build --release
```

编译后的可执行文件位于 `target/release/zomkill.exe`

## 🔧 技术实现

### 内存操作
程序使用 Windows API 进行进程内存的读写操作：
- `OpenProcess` - 打开游戏进程
- `ReadProcessMemory` - 读取内存数据
- `WriteProcessMemory` - 写入内存数据

### 内存地址
- **阳光值**: 基址 `0x00755E0C` + 偏移 `0x868` + 偏移 `0x5578`
- **冷却时间**: 基址 `0x00755E0C` + 偏移 `0x868` + 偏移 `0x15C` + 偏移 `0x70`

### 界面设计
使用 egui 框架构建响应式界面：
- 顶部面板显示标题和菜单
- 中央面板显示功能模块
- 底部状态栏实时显示运行状态

### 中文字体
- 内置 Noto Sans CJK SC (思源黑体简体中文)
- 字体文件在编译时打包进可执行文件
- 无需额外安装字体，开箱即用
- 完美显示所有中文字符
